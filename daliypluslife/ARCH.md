# daliypluslife.com — 網站架構

## 定位
生活資訊引流網站，搭配 FB 專頁做流量導入
目標受眾：台灣泛大眾，以中老年 FB 用戶為主力

## 核心內容方向
- 生活實用資訊（健康、飲食、居家、省錢技巧）
- 震驚體／好奇體標題農場文
- 懶人包、排行榜、「你不知道的 XX 件事」格式
- 節慶、時事關聯生活文
- 情感共鳴類（家庭、人際、勵志）

## 技術架構
- 平台：WordPress
- 主機：Cloudways（三站共用同一台，1GB）
- CDN：Cloudflare（含網域管理）
- FB 專頁：https://www.facebook.com/profile.php?id=61580800352029

## 流量策略
- **主要引流**：FB 專頁發文 → 導流回網站（台灣農場文主流模式）
- **FB 貼文策略**：
  - 截短文章前兩句，吸引點進連結
  - 發文時間：早 7-9 點、午 12-13 點、晚 8-10 點（台灣 FB 高峰）
  - 每日發文 3-5 篇
  - 加入「分享給家人朋友」號召，擴大觸及
- **SEO 輔助**：非核心，但文章需有基本關鍵字覆蓋
- 文章產出頻率：每日 5-10 篇（農場式批量產出）

## 變現模式
- Google AdSense（高 PV、多廣告位）
- 原生廣告（Taboola / Yahoo 原生廣告）
- （未來）電商導購聯盟

## 競品參考（同類引流農場模式）
- **GigaCircle 模式**：FB 貼文導流，高頻發文，社群裂變傳播
- **TTshow（台灣達人秀）**：懶人包＋農場文混合，標題震驚體
- **一些 LINE 群組導流站**：情感共鳴文為主，老年受眾
- 共同特徵：
  - 標題必帶情緒（震驚、感動、原來如此、你一定要知道）
  - 封面圖強烈視覺衝擊
  - 文章結構簡單，大量換行，方便手機閱讀
  - 廣告位置密集（文章中段插廣告）

## 借鑑重點
- 標題公式：「原來＿＿竟然可以＿＿！99% 的人都不知道」
- 封面圖：人臉、食物、對比圖效果最好
- 文章長度：600-1000 字，短段落，多圖
- FB 貼文：前兩行就要讓人想點，結尾加「你覺得呢？留言告訴我們」
